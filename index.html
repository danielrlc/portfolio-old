About

I'm a self-taught frontend web developer. My interest was piqued when I was creating a WordPress website for learning Finnish. I started digging into the PHP and jQuery in the plugins I'd installed, and I was soon hooked. After a while of playing around, I decided I wanted to learn how to do it properly, so I signed up for a three-month course with Thinkful. I liked the course as it threw me straight into it: building projects, using Github, engaging with the community, and having weekly mentor sessions. Since completing the course, I've been fleshing out my skills by taking other online courses, reading books and listening to many hours of tech podcasts.

I have a background in language training and live in Finland with my wife and two young daughters.

Project notes

Portfolio

I built this portfolio site (the site you're looking at now) from scratch, i.e. without any CSS framework like Bootstrap. It is fully responsive and can be viewed and read easily on any device. If you're on a laptop or desktop, you can test this by adjusting the width of your browser window. The content works effectively at all widths, and never becomes squashed or difficult to read.

John Williams quiz

I made this short quiz as simple and interactive as I could. Wrong answers turn red, sound an ominous gong and trigger feedback, which changes for each wrong answer. The right answer turns green and sounds a (mildly irritating) "ta-da" fanfare.

The quiz demonstrates my improving frontend skills in HTML, CSS, JavaScript, jQuery, usability and design. The quiz content is injected via JavaScript into the page. I created a prototype object from which all the question objects are created.

Hot or cold game

The app picks a random number between 1 and 100 and you have to guess it. With each guess, you get feedback on how close you got.

For this project, I was required to add interactivity to the HTML and CSS files I was presented with. It was instructive and useful to take over a project with some code already written by somebody else.
