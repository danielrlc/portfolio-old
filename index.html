About

I started learning frontend web development in August 2014 with Thinkful. I have learnt HTML5, CSS, JavaScript, jQuery, AJAX and RWD. My projects are on GitHub. This site (source code) and my John Williams quiz (source code) best demonstrate what I've learnt so far. Alongside my Thinkful course, I have used all these learning resources (and more) extensively to get as broad and cutting-edge an education as possible: Eloquent JavaScript, Lynda.com, Frontend Masters, Codecademy, Code School, JavaScript Jabber, JavaScript Weekly, CSS Weekly, Web Design Weekly, and Jakob Nielsen's newsletter.

I have a background in teaching and a long-time interest in digital learning and usability. I created Miten menee?, a WordPress-based website for Finnish learners, in 2013. I helped improve this flashcards plugin, by sending some code tweaks to the plugin's creator.

I completed a master's degree in Information Systems in 2011. I studied business analysis, systems analysis, relational databases and website usability.
