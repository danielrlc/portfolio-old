About

I started learning frontend web development in August 2014 with Thinkful, where I learnt through building various projects (Github links below) and weekly mentor sessions. I've learnt mainly HTML, CSS, JavaScript, jQuery and AJAX, but I've also picked up some skills in design and usability. For me, frontend web development hits the spot, as it's all about hiding technical complexity and making technology usable. I am currently making use of all the following cutting-edge resources to improve my skills further: Eloquent JavaScript, Lynda.com, Frontend Masters, Codecademy, Code School, JavaScript Jabber, JavaScript Weekly, CSS Weekly, Web Design Weekly, and Jakob Nielsen's newsletter.

I have a background in language training and live in Finland with my wife and two young daughters.

Project notes

John Williams quiz

Live quiz
Source code

I made this short quiz as simple and interactive as possible. Wrong answers turn red, sound an ominous gong and trigger feedback, which changes for each wrong answer. The right answer turns green and sounds a "ta-da" fanfare.

The quiz demonstrates my improving frontend skills in HTML, CSS, JavaScript, jQuery, usability and design. The site has responsive images, and the quiz content is injected via JavaScript. I created a prototype object from which all the question objects were created.

I am getting better at making colours work together and using white space effectively. I used Paletton.com to help me choose a suitable blue, green and red colour palette to work with.

Hot or cold game

Live app
Source code

The app picks a random number between 1 and 100 and you have to guess it. With each guess, you get feedback on how close you got.

For this project, I was required to add interactivity to the HTML and CSS files I was presented with. As a result, I learnt a very useful skill: how to take on a project that's already been started.
