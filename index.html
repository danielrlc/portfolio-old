Project notes

Portfolio

I built this portfolio site (the site you're looking at now) from scratch without using any framework. It is fully responsive and can be viewed and read easily on any device at any viewport width. I kept the responsiveness very simple with just two break points.

For good accessibility, I left wide margins when viewing the text sections in landscape view on a smart phone, so the user can zoom in considerably to enlarge the font, and still retain the full text on their screen. For anyone quite comfortable with small(ish)-sized text, I provided more compact text for portrait view on a smart phone.

John Williams quiz

I made this short quiz as simple and interactive as I could. Wrong answers turn red, sound an ominous gong and trigger feedback, which changes for each wrong answer. The right answer turns green and sounds a (mildly irritating) "ta-da" fanfare.

The quiz demonstrates my improving frontend skills in HTML, CSS, JavaScript, jQuery, usability and design. The quiz content is injected via JavaScript into the page. I created a prototype object from which all the question objects are created.

About

I'm a self-taught frontend web developer. My interest was piqued when I was creating a WordPress website for learning Finnish. I started digging into the PHP and jQuery in the plugins I'd installed, and I was soon hooked.

After playing around for a while, I realised I wanted to learn how to do it properly, so I signed up for a three-month course with Thinkful. I liked the course as it threw me straight into it: building projects, using Github, engaging with the community, and having weekly mentor sessions. Since completing the course, I've been fleshing out my skills by taking other online courses, reading books and listening to many hours of tech podcasts.

I have a background in language training and live in Finland with my wife and two young daughters.
